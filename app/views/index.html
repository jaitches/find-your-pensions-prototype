Before you start

You must set up pensions for the participant before you can use the prototype. This can be done by adding new pensions or by copying a pension from the list of example pensions. This will create a new pension and allow you to update the information.

If the provider doesn't exist in the prototype database you will need to add the pension provider before you can add any pensions that they manage.

You can manage the providers from the list of pension providers.
